What is CustQR?

Cust QR is a personal project which is built with the only motive of customizing boring QR's into awesome unique QR's.

Why CustQR?

Cust QR is built using Google's robust ZXing Library which enables hassle free user experience. Also the site aims to provide all the possible customizations which could be possible on a QR.
